What is a Closure?

  • A closure is an inner function that has access to the outer (enclosing) function’s variables—scope chain. The closure has three scope chains: it has access to its own scope (variables defined between its curly brackets), it has access to the outer function’s variables, and it has access to the global variables. - javascriptissexy.com
  • function getCounter() {
  var counterValue = 0;

  function increment() {
    counterValue += 1;
    return counterValue;
  }

  return increment;
}

var myCounter = getCounter();
myCounter(); // 1
myCounter(); // 2

What is a Promise?

  • The Promise object is used for asynchronous computations. A Promise represents an operation that hasn't completed yet, but is expected in the future. - Mozilla developer network
  • fetch('/statuses/show.json?id=2104') // Returns a Promise
